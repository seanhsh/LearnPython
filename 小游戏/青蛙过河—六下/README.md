# 青蛙过河 - 背单词游戏

一个有趣的Python背单词游戏，帮助你在游戏中学习英语单词！

## 游戏特色

- **三条河道**：每道有漂浮的荷叶，荷叶上显示英文单词
- **键盘互动**：敲击键盘上对应单词首字母的按键
- **青蛙跳跃**：答对后青蛙会跳到那片荷叶上
- **通关目标**：从左边小石头跳到右边小石头

## 游戏系统

- **3个难度关卡**：每个关卡有不同的单词库
  - 第1关：基础单词（26个）
  - 第2关：形容词（22个）
  - 第3关：抽象名词（24个）
- **计分系统**：每答对一个单词得10分
- **速度递增**：关卡越高，荷叶移动速度越快

## 安装依赖

在运行游戏之前，需要安装pygame库：

```bash
pip install pygame
```

## 运行游戏

### 方法1：使用启动脚本（推荐）
双击 `启动游戏.bat` 文件即可运行游戏。

### 方法2：直接运行Python文件
在命令行中运行：

```bash
python frog_game.py
```

## 游戏操作

- **空格键**：开始游戏 / 重新开始
- **字母键**：敲击对应单词首字母
- **P键**：暂停/继续游戏
- **ESC键**：返回主菜单

## 文件说明

- `frog_game.py` - 主游戏文件
- `words.py` - 单词库文件
- `config.py` - 游戏配置文件
- `启动游戏.bat` - Windows启动脚本
